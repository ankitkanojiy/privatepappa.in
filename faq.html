Frequently Asked Questions

Answers about PrivateVault's offline vault, encryption, permissions, media tools, document viewing, sharing controls, privacy, and security design.

Offline-first

Does PrivateVault require internet access?

No. PrivateVault is an offline-first encrypted vault. Core vault features work locally on your device without internet access.

No cloud

Does PrivateVault upload my files to cloud storage?

No. Vault files remain local unless you manually export or share them outside the app.

Local processing

Are thumbnails, previews, and search processed locally?

Yes. The app processes thumbnails, indexing, previews, and organisation locally without sending vault content to external servers.

Permissions

Which permissions does the app use?

PrivateVault uses only permissions required for selected features, such as reading user-selected images, videos, audio, documents, and optional camera capture when you use camera features.

Camera

When is camera permission requested?

Camera permission is requested only when you use direct camera capture or document scanning features.

No internet

Does the vault need INTERNET permission?

The core offline vault does not require internet permission. This helps reduce data-leak risk.

Encryption

How are files protected?

Files are stored in encrypted form inside the vault. The app uses AES-256-GCM encryption with PBKDF2 key derivation, SHA-256 integrity hashing, protected vault sessions, and chunk-level authentication. All encryption and decryption happens locally on your device.

Password

Does the app support long vault passwords?

Yes. The vault design supports strong passwords up to 127 characters.

Screenshots

Are screenshots blocked?

Yes. Screenshot and screen-recording protection is enabled for sensitive vault screens in the release build.

Integrity

How does PrivateVault reduce corruption risk?

The app includes integrity validation, transactional writes, metadata recovery, chunk checks, restore validation, and corruption-resistant encrypted file handling.

Import

Can I import many file types?

Yes. PrivateVault supports importing common media, documents, code files, and other user-selected files into the encrypted vault.

Export

Can I export decrypted files?

Yes. Files can be exported in decrypted form only when you explicitly choose to export them to a selected location.

Backup

Can I create encrypted backups?

Yes. PrivateVault can create encrypted backup files of your vault. These backups can only be opened inside the PrivateVault app using a pre-set password you choose. Backups cannot be read by other apps or tools.

Share toggle

Can file sharing be disabled?

Settings include a toggle to enable or disable external file sharing. When disabled, decrypted file sharing outside the app is blocked. This feature may be available in a future update but is not yet enabled.

Safe sharing

How does secure sharing work?

Shared files are prepared as temporary decrypted copies, passed to the selected app, and cleaned up after the sharing session. This feature may be available in a future update but is not yet enabled.

Documents

Can I view Word, Excel, PowerPoint, and PDF files?

The app supports viewing supported document and PDF files. Editing is limited. Not all formatting, layouts, or file types will display perfectly.

Password files

Can I open password-protected PDFs?

Yes. Supported password-protected PDFs can be opened after entering their password, with retry limiting and session isolation. Passwords are not stored.

Code files

Can I view and edit code files?

Yes. The app supports viewing and plain-text editing of common code formats such as JavaScript, TypeScript, C, C++, Java, PHP, Python, Kotlin, Swift, HTML, CSS, JSON, XML, YAML, and more.

Images

Can I view images?

Yes. The app supports image viewing, zooming, panning, and basic image preview. Image-to-PDF conversion is supported where available.

Video

Can videos play in portrait and landscape?

Yes. The video player supports dynamic rotation according to device rotation without forcing horizontal orientation.

Zoom

Does media support double-tap and pinch zoom?

Yes. Images and videos support double-tap zoom and pinch zoom with edge resistance and controlled panning.

Scanner

Can I scan documents with the camera?

Yes. The app includes camera document scanning with basic crop and PDF export. Scan results depend on image quality and lighting conditions.

Macros

Are document macros allowed?

No. Macro execution and active document content are blocked to reduce malicious document risk.

APK files

Can APK files be run inside the app?

No. APK files can be inspected in read-only mode, but they cannot be executed or installed by the vault.

Performance

Is PrivateVault optimised for budget phones?

The app adjusts thumbnail and preview processing based on device memory and core count. Lower-end devices receive reduced concurrent workloads.

Feature Availability Note

Some advanced features depend on device capabilities, Android version, supported file types, and production build configuration. Not every feature will be available on every device or for every file format.
